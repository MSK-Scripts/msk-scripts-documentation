# MSK EngineToggle

[GitHub Download](https://github.com/MSK-Scripts/msk_enginetoggle) | [CFX Post](https://forum.cfx.re/t/msk-enginetoggle-toggle-engine-on-off/4793840)

## Description

- The engine keeps running if you leave the vehicle without turning it off
- You can set the engine to start automatically when entering a vehicle
- Set a Command and Hotkey _(RegisterKeyMapping)_
- Set your own Notification and Progressbar
- If `Config.VehicleKeys` is set to `true`, only the Owner or someone with a Key can start the Engine
- Admin Command to start/stop the engine without a Key _(Bypass)_
- Whitelist for models and plates that do not need a Key
- Save the Angle of the Steering Wheel _(Synced)_
- Lockpick/Hotwire Function in compatibility with VehicleKeys
