# Admin Commands

:::danger
You need **msk_core**! — [Download it here](https://github.com/MSK-Scripts/msk_core)
:::

| Command | Description |
|---|---|
| `/adcuff <playerID>` | Cuffs the player with specified ID |
| `/adhardcuff <playerID>` | Hardcuffs the player with specified ID |
| `/aduncuff <playerID>` | Uncuffs the player with specified ID |
| `/adankletracker <playerID>` | Activates/Deactivates the Ankletracker for specific ID |
| `/adheadbag <playerID>` | Activates/Deactivates the Headbag for specific ID |
| `/adtape <playerID>` | Activates/Deactivates the Tape for specific ID |
