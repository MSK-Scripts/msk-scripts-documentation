## Description

- Cuff and Uncuff Players with Animations and Props
- Hardcuff a Player _(Player can't move)_
- Drag and Undrag Player
- Put Player in/out of vehicle
- AnkleTracker — _(Specific job can track a players position)_
- Headbag — _(Player gets a Blackscreen and a Bag on their head)_
- Tape — _(Player gets tape on their mouth and can't talk)_

---

- Cuff, uncuff, etc. players with items, commands or via export
- Timer for automatic uncuff
- Admin commands
- Discord Logs
- All data saved in `database.json`
- Cuff Status, Ankletracker, etc. saved to database and restored after relog

- Example implementation for esx_policejob
- Example implementation for Jaksam job_creator
