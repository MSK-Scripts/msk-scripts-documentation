# MSK Banking

[**Tebex [16.99€ inkl. MwSt.]**](https://www.msk-scripts.de/package/5118928) — _Encrypted_  
[**Tebex [29.99€ inkl. MwSt.]**](https://www.msk-scripts.de/package/5302445) — _Only Serverside Encrypted_

[**YouTube Video Preview**](https://youtu.be/tZTx9l5vr_w) | [**CFX Post**](https://forum.cfx.re/t/esx-msk-banking-advanced-bankingsystem/4859560)

## Description

- Create your first main account and a second account
- Create society accounts _(integrated System or Addon_Account)_
- Invoices, Investment, Transaction history
- Animation at the ATMs
- Deposit, Withdraw and Transfer money
- Transfer money via Accountnumber _(offline Transfer)_
- NPCs or Marker at the Bank
- Bankcard item needed to access an ATM
- Set different bank locations with configurable options
- Set ranks that have access to the society account
- Set your own Notification and Custom Banner
- Discord Logs

## Requirements

- [ESX Legacy](https://github.com/esx-framework/esx_core)
- NativeUI
- [msk_core](https://github.com/MSK-Scripts/msk_core)
- [oxmysql](https://github.com/overextended/oxmysql)

- esx_billing
