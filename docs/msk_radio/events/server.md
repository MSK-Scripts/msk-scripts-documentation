# Server Events

## Open Radio

**Parameters**  
**playerId** - The ID of the Player

```lua
TriggerClientEvent('msk_radio:openRadio', playerId)
```

## Close Radio

**Parameters**  
**playerId** - The ID of the Player

```lua
TriggerClientEvent('msk_radio:closeRadio', playerId)
```
