# Discord Multi Bot

A modern, self-hosted Discord multi bot built on **Discord.py**, full feature set out of the box.

## What to expect

- **Events** — Event Logging
- **Commands** — Admin Commands, App commands and many more
- **Minigames** — Dice, Flipcoin, TicTacToe, Handman, Rock Paper Scissors, Wordle, Connect4, Blackjack, Slots
